# Jump Game(Leetcode 55)

Category	Difficulty	Likes	Dislikes
algorithms	Medium (31.45%)	2645	251
Tags
array | greedy

Companies
microsoft

Given an array of non-negative integers, you are initially positioned at the first index of the array.

Each element in the array represents your maximum jump length at that position.

Determine if you are able to reach the last index.

Example 1:

Input: [2,3,1,1,4]
Output: true
Explanation: Jump 1 step from index 0 to 1, then 3 steps to the last index.
Example 2:

Input: [3,2,1,0,4]
Output: false
Explanation: You will always arrive at index 3 no matter what. Its maximum
             jump length is 0, which makes it impossible to reach the last index.

# 尝试用递归recursion

  1. 思路
当 当前未知的值和下标的和大于vector大小的时候，一定到达；
其他情况， 遍历后续数据，找到可以作为下个开头的起始位置，条件下标和值的和大于起始值；
向量丢弃无用数据，复制新的开头到最后的vector数据
循环终止条件，递归返回值为true
  1. 代码
class Solution {
  public:
    bool canJump(vector<int>& nums) {
      if(nums.size()==0) return false;
      int cur = nums[0];
      if (cur == 0 ) return false;
      int size = nums.size() - 1;
      cout<< "size:"<<size<< " cur:"<<cur<<endl;
      if ((nums[0] +1)  >= nums.size()) {
        return true;
      } else{
        for(int j = 1; j<= nums[0];j++) {
          if(nums[j]+j <= nums[0]) continue;
           nums.erase(nums.begin(),nums.begin()+j);
          if(canJump(nums)) {return true;}
        }
        return false;
      }
    }
};

# 贪心greedy

  1. 思路
小区间遍历
三个下标， headIndex， curIndex, endIndex
到达条件 endIndex + 1 > size
  1. 代码
class Solution {
public:
    bool canJump(vector<int>& nums) {
      int size = nums.size();
      if(nums.size() == 0) return false;
      if(nums.size() == 1) return true;
      int curIndex = 0; 
      int headIndex =0;
      int EndIndex = headIndex + nums[headIndex];
      if (EndIndex == 0) return false;
      if (EndIndex + 1>= size) return true;
      for(curIndex;curIndex<=EndIndex;curIndex++) {
        if(curIndex + (nums[curIndex] ) > (headIndex + nums[headIndex] )){
          headIndex = curIndex;
          EndIndex = headIndex + nums[headIndex];
          if(EndIndex + 1 >= size) return true; 
        }
      }
      return false;
    }
};
