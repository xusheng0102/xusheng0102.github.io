# Alibaba Cloud VPS build SSR super detailed tutorial + ShadowsocksR client download

  1. Recommended vps: Alibaba Cloud's Simple Application Server
<https://www.aliyun.com/product/swas?spm=5176.12825654.eofdhaal5.35.3dbd2c4aJTjew1&aly_as=iTNTTJuPT>
  1. ssr build tutorial My Environment: CentOS 7.3
$ wget --no-check-certificate -O shadowsocks-all.sh https://raw.githubusercontent.com/teddysun/shadowsocks_install/master/shadowsocks-all.sh
$ chmod +x shadowsocks-all.sh
$ ./shadowsocks-all.sh 2>&1 | tee shadowsocks-all.log

Three lines of code can be executed line by line, and then fill in the account password. After execution, you will be prompted to enter the source language, password, port, and encryption method. (I use 1314 for the port here; the source language is Go; the encryption method is aes-256-cfb;) The following effect will eventually appear, indicating that the deployment is complete:

Congratulations, your_shadowsocks_version install completed!
Your Server IP :you public network IP
Your Server Port :1314
Your Password :123456
Your Encryption Method:aes-256-cfb

Welcome to visit:https://teddysun.com/486.html
Enjoy it
  1. modify the firewall configuration Open the Configuration file:
sudo vi /etc/sysconfig/iptables-config

Insert the following sentence:

- A the INPUT - P TCP - m State - State NEW - m TCP - dport 1313 - J ACCEPT

And you can see the iptables rules by

netstat -tlnp
  1. add FireWall rule in your simple Application Sercer controll page
Safety --> FireWall --> Add rule -->
types customize 
protocol TCP
portRange 1314
  1. if you want uninstall shadowsocks
./shadowsocks-all.sh uninstall
  1. 刪除阿里雲盾

  1. shadowsock的使用命令
    启动：/etc/init.d/shadowsocks-go start
    停止：/etc/init.d/shadowsocks-go stop
    重启：/etc/init.d/shadowsocks-go restart
    查看状态：/etc/init.d/shadowsocks-go status

or

 sudo /usr/bin/shadowsocks-server -u -c /etc/shadowsocks-go/config.json
